
Este arquivo explica de forma direta e objetiva o propósito do seu projeto visual (Front-end), detalhando:
1. **O objetivo**: Mostrar que ele serve como a camada visual/cliente para interagir com a `JobsAPI`.
2. **Tecnologias**: Uso de `HttpClient` e manipulação de JSON.
3. **Funcionalidades**: O mapeamento visual das ações do CRUD (Listar, Buscar, Cadastrar, Editar e Deletar).
4. **Instruções de Execução**: Lembrete de que a API precisa estar rodando em segundo plano e como iniciar o app.
